# AdaptivePN

Распределение эффективного показателя преломления в сечении PN-фазовращателе (прикладное напряжение: 1e-4 V, acceptor density = 6e15 1/m^3, donor density = 2.5e7 1/m^3, оксид кремния)

Neff_max = 3.48
